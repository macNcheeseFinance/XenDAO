What is Wallet Minter

Wallet minter is a tool that allows you to generate as many EVM(Ethereum, BSC, polygon,...) compatitible addresses as you wish. XEN contract allows 1 concurrent claim per each wallet address. This allows you to claim XEN from hundreds of wallets at once.

Addresses are generated in your browser. They are not stored anywhere.
It is 💀EXTREMELY IMPORTANT💀 that you manually SAVE the accounts on your device or else they will be IRREVERSIBLY lost FOREVER.

Earn Xen DAO(XD) token

You receive XD token for each wallet generated(during mass send).
Xen DAO token(XD) can be staked. All of the protocol fees are distributed to XD stakers.


How does it work?

1.) Wallets are generated in your browser through etherJS

2.) Tool mass sends native token(ETH/BNB/MATIC/....) to ALL new addresses from your "metamask" wallet
You will receive bonus Xen DAO(XD) tokens minted to your wallet
(each address needs enough gas to claim + mint + transfer XEN)

3.) Tool loops through all of the wallets and claims XEN
(it signs the transactions programatically using the generated private key)

4.) Once your term matures, tool allows you to loop through all wallets and mint the tokens accross all wallets at once

5.) Tool allows you to mass send XEN balances from all accounts into a single address

6.) That's it. XEN from all accounts is now collected in your single wallet. You can now sell or stake XEN.


Is it Safe to use?

The tool is safe to use. There is no backdoor, however theoretically any javascript file on the site could log the wallets and send them to the server. If you don't trust, you should use CLEAN VERSION, where source code can be viewed and verified. You can even save it as .html and use locally on your device.
